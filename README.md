# Windows-Cangjie-Updater
# 🌟 Windows 內建倉頡輸入法 (三代/五代) 碼表自動更新工具

這是一個使用 PowerShell 撰寫的自動化腳本，旨在解決 Windows 10 / Windows 11 內建倉頡輸入法「選字排序不佳」、「缺字」或「預設非台灣/香港常用字體」等痛點。

透過此腳本，您可以一鍵將系統預設的倉頡碼表替換為社群優化版本（支援倉頡三代與五代），並自動開啟系統 HKSCS（香港增補字符集）支援。

## ✨ 核心特色

* 🚀 **一鍵全自動**：自動處理下載、解壓縮、關閉輸入法進程、替換系統檔與修改登錄檔。
* 🛡️ **安全自動升權**：執行時會自動檢查並請求系統管理員權限（UAC），無需手動複雜操作。
* 🗂️ **互動式選單**：提供多種碼表偏好版本（台灣習慣、香港習慣、一般排序等），支援 10 秒倒數自動選擇預設值。
* 🔄 **安全備份機制**：在替換任何檔案前，會自動將原系統碼表備份至 `System32\zh-hk\Backup_YYYYMMDD` 目錄下，安全有保障。
* 🌐 **完美處理中文路徑**：全腳本採用 UTF-8 (BOM) 編碼與 PowerShell 原生物件比對，解決傳統 `.bat` 處理中文路徑容易亂碼的問題。

## 🛠️ 使用教學 (快速開始)

不需要任何程式基礎，請依照以下步驟操作：

1. **下載腳本**：
   * 點擊網頁右上角的 `Code` 綠色按鈕 -> 選擇 `Download ZIP`。
   * 解壓縮後，根據你的需求進入 `Cangjie3` 或 `Cangjie5` 資料夾。
2. **執行腳本**：
   * 找到你要執行的腳本（例如 `AutoUpdate-CJ5.ps1`）。
   * **對著檔案按「右鍵」**，選擇 **「使用 PowerShell 執行 (Run with PowerShell)」**。
3. **授權管理員權限**：
   * 系統會跳出使用者帳戶控制 (UAC) 視窗詢問是否允許變更，請點選 **「是」**。
4. **選擇版本**：
   * 在彈出的藍底黑字視窗中，輸入對應的數字選擇你想要的碼表版本（若 10 秒未輸入，將自動選擇預設的「台灣用字習慣」或「2004+版本」）。
5. **重啟電腦**：
   * 腳本顯示「更新成功」後按任意鍵關閉視窗。
   * ⚠️ **請務必重新啟動電腦**，新的碼表與選字順序才會完全生效！

### ⚡ 傳統批次檔 (.bat) 快速執行版 (推薦台灣使用者)

如果您不想透過介面選單操作，或者偏好使用傳統的命令提示字元 (CMD)，本專案每個資料夾內亦保留了傳統的 `.bat` 批次檔（例如 `AutoUpdateCJ5.bat`）。

* **特點**：跳過所有倒數與互動選單，**一鍵全自動安裝**。
* **寫死預設版本**（專為台灣用字習慣配置）：
  * **倉頡三代**：固定提取 `Windows 10 2004及之后的Windows` 版本。
  * **倉頡五代**：固定提取 `傳統漢字優先（偏好台灣用字習慣）` 版本。
* **使用方式**：
  對著 `.bat` 檔案按 **右鍵** -> 選擇 **「以系統管理員身分執行」** 即可自動完成備份與更新。

## ⚠️ 常見問題與注意事項

### 🛠️ 疑難排解：解決「一閃而過」或「紅字報錯」

* **Q：執行時視窗一閃而過，或是出現「禁止執行指令碼」的紅字報錯？**

* **A：** 這是因為 Windows 基於安全考量，預設鎖定了 PowerShell 腳本的執行權限。請依照以下步驟簡單解鎖：
 1. 按下鍵盤 **`Win + X`** 鍵，選擇 **「Windows PowerShell (系統管理員)」** 或 **「終端機 (系統管理員)」**。
 2. 複製並貼上以下指令，然後按 **Enter**：
    ```powershell
    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
    ```
 3. 系統詢問時，輸入 **`Y`** 並按 **Enter** 確認。
 
 完成後，再次對著腳本按 **「右鍵 -> 使用 PowerShell 執行」** 即可正常運作！
* **Q: 發生防毒軟體阻擋？**
  * A: 因為腳本會修改系統核心目錄 (`C:\Windows\System32`) 與登錄檔，部分嚴格的防毒軟體可能會跳出警告。此腳本原始碼完全公開透明，請安心放行。
* **Q: 更新後我想還原怎麼辦？**
  * A: 腳本執行時會在 `C:\Windows\System32\zh-hk\` 內建立名為 `Backup_年月日` 的資料夾。你需要手動將裡面的 `ChtCangjieExt.lex` 等檔案複製並覆蓋回 `zh-hk` 目錄下即可還原。
